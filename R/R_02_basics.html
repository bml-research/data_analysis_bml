Some very basics of R

Although we will try to avoid base R as much as possible, you will need a bit of basics.

We can assign a variable in R as follows:

x <- 42
print(x)
## [1] 42

The very basic data structure of R is the vector.

It may contain whole numbers:

my_nums <- c(1, 2, 3, 4)
print(my_nums)
## [1] 1 2 3 4

Or decimals:

my_dec <- c(0.1, 0.2, 0.3, 0.4)
print(my_dec)
## [1] 0.1 0.2 0.3 0.4

Or text:

my_txt <- c("Hello", "World")
print(my_txt)
## [1] "Hello" "World"

Or Booleans:

my_bools <- c(TRUE, FALSE)
print(my_bools)
## [1]  TRUE FALSE

With a shorthand:

my_bools <- c(T, F)
print(my_bools)
## [1]  TRUE FALSE

But what we will use is the data frame. See it as a bunch of column stitched together:

my_df <- data.frame(my_nums, my_dec)
print(my_df)
##   my_nums my_dec
## 1       1    0.1
## 2       2    0.2
## 3       3    0.3
## 4       4    0.4

Both the c and data.frame are functions in R. You can call them using function.name() and pass data to it as arguments.

So in the above code the vectors assigned to the variable names my_nums and my_dec are transferred as arguments in the data.frame function. This function returns a data frame object that is assigned to the my_df variable name.

Installing libraries

You can install the tidyverse library using:

install.packages("tidyverse")

If you encounter problems with dependencies try:

install.packages("tidyverse", dependencies = TRUE)

Loading packages:

You can load a package as follows:

library(tidyverse)

Make a tibble

Now we can make a fancy type of data frame from the tidyverse library:

my_tibble <- tibble(my_df)
my_tibble
## # A tibble: 4  2
##   my_nums my_dec
##     <dbl>  <dbl>
## 1       1    0.1
## 2       2    0.2
## 3       3    0.3
## 4       4    0.4

Tibbles do have some advantageous over base R data frames such as: - Tibbles only print the first 10 rows and all the columns that fit on a screen. - Each column displays its data type.

As we are using the tidyverse library during this course, we will use the tidyverse main data type: The Tibble.
